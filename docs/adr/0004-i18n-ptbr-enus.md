# ADR 0004 - i18n PT-BR and EN-US

## Status

Accepted

## Context

O projeto tem audiencia local e internacional, e precisa ser demonstravel em portfolio publico.

## Decision

Adotar i18n desde a fundacao com arquivos JSON para `pt-BR` e `en-US`.

## Consequences

- Pro: experiencia bilingue e ampliacao de alcance.
- Pro: evita retrabalho de internacionalizacao tardia.
- Con: exige manutencao de paridade de chaves.
