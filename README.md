EB2-NIW Readiness Studio

EB2-NIW Readiness Studio is a local-first web application built with Python, Flask and SQLite to help professionals organize evidence, gaps, tasks, roadmap, technical authority initiatives, and AI-assisted preparation workflows related to EB-2/NIW readiness.

The project is designed as an educational and organizational tool. It does not provide legal or immigration advice, does not estimate approval chances, and should not replace consultation with a qualified immigration attorney.

Main Goals
Organize professional evidence for EB-2/NIW readiness.
Track gaps, roadmap tasks, recommenders, publications and public portfolio initiatives.
Support PT-BR and EN-US internationalization.
Provide optional AI assistance through pluggable providers such as Mock, OpenAI, Azure OpenAI and Ollama.
Keep sensitive data local by default using SQLite.
Serve as a public technical portfolio project demonstrating Python, Flask, SQLite, local-first architecture, AI provider abstraction, i18n, testing and spec-driven development.
Legal Disclaimer

This tool is for educational and organizational purposes only. It is not legal advice, immigration advice, or a substitute for professional evaluation by a qualified immigration attorney.

Privacy Notice

Do not commit real personal data, immigration documents, private evidence, API keys, or sensitive files to this repository. Use fake/demo data for public portfolio purposes.
